# Assets

This directory is reserved for Electron packaging resources such as:

- Linux icon sets
- Windows `.ico` assets
- installer background or metadata assets if they are added later

It includes a minimal placeholder icon set:

- `icon.svg` as the editable source
- `icons/*.png` for Linux packaging
- `icon.ico` for Windows packaging

These assets are intentionally simple and can be replaced later without changing the packaging workflow.
